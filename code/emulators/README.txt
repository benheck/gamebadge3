SUPPORTED ROM FORMATS
=======================
Commodore 64: 	.PRG
Atari 800:	.ROM
Atari 2600:	.BIN
Atari 5200:	.BIN
Gameboy:	.GB
Odyssey:	.BIN
Coleco Vision:	.ROM

KNOWN ISSUES
===============
Commodore 64:
- Not all games load
- Some games freeze during game play
- Only .PRG files are supported

Atari 2600:
- No audio
- No display scaling

Atari 5200:
- Only .BIN files are supported
- Some games will not load (freezes at Atari logo screen)

Gameboy:
- No audio
- Not frame accurate
- Some games cause hefty lag
